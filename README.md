## 🎮 Режимы Управления

### Режим 0️⃣ - РУЧНОЙ КОНСТРУКТОР (Constructor Mode)
**Описание:** Пошаговое управление каждым пальцем с полной защитой от повреждений.

**Особенности:**
- Графический интерфейс с ползунками для каждого пальца
- Выбор угла сжатия (0-180°)
- Очередь команд с возможностью редактирования
- Добавление паузы между движениями
- **ПОЛНАЯ ЗАЩИТА ОТ ФАКА**

---

### Режим 1️⃣ - БИО-ЭМГ ИНТЕРФЕЙС (Electromyography Interface)
**Описание:** Управление кистью через датчики ЭМГ на руке пользователя.

**Особенности:**
- Чтение биоэлектрических сигналов с предплечья
- Интуитивное управление (мышечные сокращения = движения пальцев)
- **ПОЛНАЯ ЗАЩИТА ОТ ФАКА** (как в конструкторе)

**Статус:** 🔧 В разработке

---

### Режим 2️⃣ - АВТОНОМНЫЙ ХВАТ (Autonomous Grasp)
**Описание:** Автоматическое сжатие кисти в ответ на сигнал датчика расстояния.

**Особенности:**
- Датчик расстояния определяет наличие предмета
- При обнаружении предмета кисть автоматически сжимается
- При отсутствии предмета кисть открывается
- **БЕЗ ОГРАНИЧЕНИЙ** (полное сжатие/разжатие)

**Датчик:** Ультразвуковой HC-SR04 (GPIO 34, 35)

---

### Режим 3️⃣ - ГОЛОСОВОЙ МОДУЛЬ (Voice Control)
**Описание:** Управление кистью голосовыми командами через браузер.

**Поддерживаемые команды:**
- "кулак" / "сожми" → сжать все пальцы (180°)
- "ладонь" / "открой" → разогнуть все пальцы (0°)

**Особенности:**
- Использует Web Speech API браузера
- Распознавание русской речи
- **ПОЛНАЯ ЗАЩИТА ОТ ФАКА** (как в конструкторе)

## 🛡️ Система Защиты от Повреждений (Anti-Jam System)

### Запрещённые Положения Руки

Эти положения **ЗАПРЕЩЕНЫ** на уровне электроники для предотвращения поломки суставов среднего пальца:

```
❌ ПОЗИЦИЯ 1: Отогнут ТОЛЬКО средний палец
   └─ Все остальные согнуты (180°)
   └─ ПОЧЕМУ: Средний палец не может быть один открыт

❌ ПОЗИЦИЯ 2: Отогнуты ТОЛЬКО средний + большой пальцы
   └─ Указательный, безымянный, мизинец согнуты (180°)
   └─ ПОЧЕМУ: Анатомическая ограничение суставов
```

### Правило 1️⃣ - Полностью Сжатая Рука (ВСЕ = 180°)

При попытке отогнуть палец из полностью сжатого состояния:

✅ **РАЗРЕШЕНО:**
- Отогнуть любой палец, если будут отогнуты ≥2 других пальца
- Пример: открыть большой + мизинец → можно открыть средний ✓

❌ **ЗАПРЕЩЕНО:**
- Отогнуть ТОЛЬКО средний палец
- Отогнуть ТОЛЬКО средний + большой пальцы

### Правило 2️⃣ - Полностью Разогнутая Рука (ВСЕ = 0°)

При попытке согнуть палец из полностью разогнутого состояния:

❌ **ЗАПРЕЩЕНО:**
- Согнуть все пальцы кроме среднего
- Согнуть указательный/безымянный/мизинец, оставляя средний + большой разогнутыми

### Применение Защиты

| Режим | Конструктор | ЭМГ | Автохват | Голос |
|-------|:-----------:|:---:|:--------:|:-----:|
| Защита от фака | ✅ ВКЛЮЧЕНА | ✅ ВКЛЮЧЕНА | ❌ ВЫКЛЮЧЕНА | ✅ ВКЛЮЧЕНА |
| Полная свобода | ❌ НЕТ | ❌ НЕТ | ✅ ДА | ❌ НЕТ |

GND ──────────────► GND

```

GND ───────────────► GND
```

## 🚀 Загрузка Кода

### Arduino UNO:

1. Установите Arduino IDE
2. Установите библиотеку `Adafruit PCA9685`
3. Загрузите файл `arduino/servo_controller.ino`
4. Выберите Board: `Arduino Uno`
5. Нажмите Upload

### ESP32:

1. Установите Arduino IDE
2. Добавьте поддержку ESP32 (Preferences → Additional Boards Manager URLs)
3. Установите пакет `esp32` v2.0.0+
4. Загрузите файл `esp32/main.ino`
5. Выберите Board: `ESP32 Dev Module`
6. COM порт и остальные параметры установите по умолчанию
7. Нажмите Upload

## 📊 Протокол I2C Команд

### Arduino Slave Address: `0x08`

#### Формат команды:

```
BYTE 0: Палец (0-5)
├─ 0 = ВСЯ КИСТЬ (все пальцы вместе)
├─ 1 = БОЛЬШОЙ ПАЛЕЦ
├─ 2 = УКАЗАТЕЛЬНЫЙ
├─ 3 = СРЕДНИЙ
├─ 4 = БЕЗЫМЯННЫЙ
└─ 5 = МИЗИНЕЦ

BYTE 1: Угол (0-180 градусов)
├─ 0° = полностью разогнут
├─ 90° = полусогнут
└─ 180° = полностью согнут
```

#### Примеры команд:

```cpp
// Согнуть все пальцы (кулак)
Wire.beginTransmission(0x08);
Wire.write(0);      // ВСЯ КИСТЬ
Wire.write(180);    // 180°
Wire.endTransmission();

// Разогнуть указательный палец
Wire.beginTransmission(0x08);
Wire.write(2);      // УКАЗАТЕЛЬНЫЙ
Wire.write(45);     // 45°
Wire.endTransmission();
```

## 🔌 REST API Endpoints (ESP32)

| Endpoint | Метод | Параметры | Описание |
|----------|-------|-----------|---------|
| `/` | GET | — | Веб-интерфейс |
| `/set_mode` | GET | `m=0..3` | Переключить режим |
| `/execute` | GET | `f=0..5, a=0..180` | Выполнить команду |

### Примеры запросов:

```bash
# Режим конструктор
curl "http://10.10.10.1/set_mode?m=0"

# Согнуть указательный на 90 градусов
curl "http://10.10.10.1/execute?f=1&a=90"

# Открыть все пальцы
curl "http://10.10.10.1/execute?f=5&a=0"
```
